Cấu trúc thư mục gồm folder:
	+ Source: chứa code của project
	+ Document: Chứa báo cáo cho project
	+ Release : Chứa file thực thi .jar và .exe
	+ Database: Chứa file 2 file .mdf và .LDF.

Hướng dẫn sử dụng:
1. Cài đặt SQL Server.
2. Setup password user "sa" là "123456"
3. Mở SQL Server Management Stdio, rồi Attach database trong folder database vào.
4. Sau đó có thể chạy phần mềm bằng file .jar hoặc .exe
5. Đang nhập user "admin", password "admin"
